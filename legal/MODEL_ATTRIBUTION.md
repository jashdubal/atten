# Kokoro-82M model attribution

Atten bundles the Kokoro-82M model, configuration, and the voice packs exposed
by its user interface.

- Upstream: <https://huggingface.co/hexgrad/Kokoro-82M>
- Pinned revision: `7884269d6fd3f9beabc271b6f1308e5699281fa9`
- License: Apache License 2.0
- Model card and authorship information: the upstream repository above

The release build records SHA-256 digests for every included model and voice
file in `Contents/Resources/Models/Kokoro-82M/MODEL_MANIFEST.json`. Atten does
not modify model weights or voice packs.
